Important Guidelines
  • Subject must already exist in the system.
  • Correct Answer must be: A, B, C, D, or E
  • Allowed Source values: PYQ, EDUSETIN, OTHER
  • Year is required only when Source = PYQ.
  • One question per row.
  • Do not modify column names.
  • Use the provided sample template.
  • New: Use Question Image Name, Option A Image Name … columns to auto-attach images from the Media Library by exact filename. If the asset is not found, the question is still imported and shown in Media Pending.
